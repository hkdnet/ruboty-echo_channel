# Ruboty::EchoChannel

A plugin for Ruboty which allows your bot to echo to the other channel.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-echo_channel'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-echo_channel
